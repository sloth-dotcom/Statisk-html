6 · 30 / 60 / 90-day rollout

Build in ROI order — dunning and onboarding first, behavioural polish last.

Days 0–30
Foundations & first flows
  1. Complete SendGrid domain authentication, link branding and the dedicated marketing subdomain.
  2. Create all custom fields; wire application events to the Marketing Contacts API.
  3. Build the core segments and the suppression groups.
  4. Build the base Dynamic Template and the Flow 1 email templates.
  5. Launch Flow 1 (Onboarding) and Flow 2D dunning — fastest payback.
  6. Enable the Event Webhook and stand up the weekly deliverability report.
Dunning recovers revenue you've already earned — pays for itself in week one.
Days 31–60
Conversion & retention
  1. Build and launch Flow 2A (trial-to-paid) and the Flow 2D renewal track.
  2. Build and launch Flow 2C (credit replenishment) and Flow 3A (dormant rescue).
  3. Launch Flow 4B (weekly new-asset email) — also the holding cadence.
  4. Start A/B test #1 (subject lines) on Flows 1 and 2A.
  5. Run the first monthly review: flow conversion vs targets.
Days 61–90
Behavioural, optimisation & scale
  1. Build and launch the behavioural flows: 4A, 4C and 4D (abandonment + cross-sell).
  2. Build and launch Flow 2B (upsell), Flow 3B (win-back) and Flow 3C (sunset).
  3. Launch Flow 4F and 4G (milestones + referral).
  4. Populate the seasonal calendar (4E) for the next two quarters.
  5. Run the quarterly review: re-baseline benchmarks against your own data.
