6 · 30 / 60 / 90-dages udrulning

Byg i ROI-rækkefølge — dunning og onboarding først, adfærds-polering til sidst.

Dag 0–30
Fundament & første flows
  1. Færdiggør SendGrid domain authentication, link branding og det dedikerede marketing-subdomæne.
  2. Opret alle custom fields; wire applikations-events til Marketing Contacts API.
  3. Byg kerne-segmenterne og suppression-grupperne.
  4. Byg den base Dynamic Template og Flow 1 email-templates.
  5. Lancer Flow 1 (Onboarding) og Flow 2D dunning — hurtigste payback.
  6. Aktivér Event Webhook og stable den ugentlige leveringsrapport på benene.
Dunning recoverer omsætning du allerede har tjent — betaler sig selv hjem i uge ét.
Dag 31–60
Konvertering & fastholdelse
  1. Byg og lancer Flow 2A (trial til betalt) og Flow 2D fornyelses-sporet.
  2. Byg og lancer Flow 2C (credit-genopfyldning) og Flow 3A (dvalende-redning).
  3. Lancer Flow 4B (ugentlig nye-asset email) — også holding-kadence.
  4. Start A/B-test #1 (subject lines) på Flow 1 og 2A.
  5. Kør den første månedlige review: flow-konvertering vs. mål.
Dag 61–90
Adfærd, optimering & skala
  1. Byg og lancer adfærds-flows: 4A, 4C og 4D (abandonment + cross-sell).
  2. Byg og lancer Flow 2B (upsell), Flow 3B (win-back) og Flow 3C (sunset).
  3. Lancer Flow 4F og 4G (milestones + henvisning).
  4. Udfyld sæsonkalenderen (4E) for de næste to kvartaler.
  5. Kør kvartalsvise review: re-baseline benchmarks mod dine egne data.
